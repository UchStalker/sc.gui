# Examples and demos

The repository contains a main demonstration:

- [demo.lua](https://github.com/UchStalker/sc.gui/tree/main/docs/examples/demo.lua) — shows the full widget set, layout managers, variables, focus, and rendering loop using regular text mode.

## Running the demos

1. Open the desired script in the project root.
2. Ensure the sc.gui is installed.
3. Run the script from your ScrapComputers environment.

## Example checklist

- Buttons and labels
- Entries and sliders
- Progress bars and list boxes
- Frames with pack and grid layouts
