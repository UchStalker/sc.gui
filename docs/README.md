# sc.gui Documentation

sc.gui is a lightweight GUI framework for ScrapComputers that provides widgets, layout managers, variables, input handling, and rendering for 2D displays.

## Recommended reading order

1. Start with [installation](getting-started/installation.md) and [quick start](getting-started/quick-start.md).
2. Understand the [application model](concepts/application-and-frames.md).
3. Learn the available [widgets](widgets/basic-widgets.md) and [layout options](layout/geometry.md).
4. Explore [variables](state/variables-and-events.md) and the example demos.
